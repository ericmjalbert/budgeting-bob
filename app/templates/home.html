Instructions

This application is meant to be the main interface with our budgetting system.

There are 3 main components to this:

  • Budgets
    This lists how much money has currently been spent for each budgeting category (grocery, clothing, games, etc...). It should be used as a reference to whether we are currently overbudget on certain spendings.
  • Transactions
    This lists all the known transactions that were copied from each bank API. From this screen manual budget categories can be applied to unknown transactions (or corrections can be applied).
  • Account Totals
    This page lists all our accounts (savings, credit, debts, etc.) and the current amount of money they contain based on the logged transactions.
